# 00101 省应对气候变化“十五五”规划

(送审稿)

为深入推进00101省“十五五”时期应对气候变化工作，如期实现碳达峰目标，依据《00101省国民经济和社会发展第十五个五年规划纲要》《美丽00101建设“十五五”规划》《00101省“十五五”碳达峰行动方案》，制定本规划。

# 一、总体目标

到2030年，全省应对气候变化工作不断深入，如期实现碳达峰目标，单位地区生产总值二氧化碳排放较2025年下降  $17\%$  ，工业领域主要排放行业有序纳入全国碳排放权交易市场，碳市场覆盖行业单位产品二氧化碳排放较2025年持续下降，温室气体自愿减排项目开发和交易取得突破，非二氧化碳温室气体管控能力进一步加强，产品碳足迹管理体系逐步完善，适应气候变化工作更加深入，全民低碳素养与对外合作水平持续提升。到2035年，全省气候治理综合实力显著增强，为实现碳中和夯实根基。

# 二、推进重点领域二氧化碳减排增汇提质

# (一)统筹推进绿色低碳转型发展

严控碳排放总量与强度，加快新型能源体系建设，合理管控化石能源消费、推动煤炭和石油消费达峰；扩容风电、光伏等非

化石能源，推进煤炭消费减量替代，拓展“外电入皖”通道，依托可调负荷等资源完善新型电力系统。优化产业布局，落实温室气体排放生态环境影响评价、节能与碳排放审查，严控“两高”项目、淘汰落后产能。聚力智能网联新能源汽车、新一代信息技术等新兴产业集群，完善绿色供应链，分级打造国家级、省级零碳园区与零碳工厂。推进城乡建设低碳升级，有序实施既有居住建筑节能改造，新建民用建筑严格执行节能率  $75\%$  标准，推广建筑光伏一体化，优化建筑电气化用能，建设绿色低碳城乡。优化运输结构，推广应用新能源清洁能源营运车船，打造内河合裕线绿色低碳运输线路、G30连霍高速等零碳公路运输通道，提升绿色集疏运比重，2030年重点行业清洁运输占比达到  $85\%$  。

# (二)系统开展生态治理固碳增汇

坚持山水林田湖草沙一体化保护和系统治理，强化皖西大别山区、皖南山区等生态屏障建设，深入推进长江、淮河、新安江、江淮运河等生态廊道建设，加强皖北平原地区农田生态系统与河网、林网建设。以大别山区、皖南山区等核心生态功能区为重点，科学开展大规模国土绿化，实施森林质量精准提升工程。强化湿地保护修复，重点开展升金湖、菜子湖等重要湿地保护修复工程。统筹实施长江流域和淮河生态带国土绿化示范工程，重点推进马鞍山、宣城长江流域示范项目以及淮南、蚌埠、滁州淮河中游至江淮分水岭示范项目建设。加强生态系统碳汇监测、核算和监管

能力建设，推动开展全省碳汇本底调查、碳储量及潜力评估。

# (三)持续深化减污降碳协同创新

深化多领域多层次减污降碳协同创新。巩固淮南市、合肥高新区等国家级减污降碳协同创新试点建设成效，稳步推进安庆市大观区生态环境分区管控减污降碳协同试点工作。遴选典型城市打造新一轮省级减污降碳协同试点，培育形成一批可复制、可推广的减污降碳绿色发展示范样板。

# 三、加强非二氧化碳温室气体管控

# (四)统筹甲烷多源综合治理

推进煤矿企业持续做好瓦斯抽采利用工作，不断提升矿区低浓度瓦斯综合利用能力和水平。推广稻田秸秆高效腐熟技术，源头压降稻田甲烷产生量。持续提升畜禽粪污资源化利用水平，降低养殖领域甲烷逸散。开展生活垃圾填埋场综合整治，提升填埋气收集回收利用率。统筹推进市政污水管网更新改造，消减污水处理及污泥处置等环节甲烷排放。支持具备条件的地市布局甲烷控排专项试点。

# (五)加强氧化亚氮排放控制

推动硝酸生产企业加装催化分解装置，削减工艺尾气中氧化亚氮，鼓励有条件企业开展尾气提纯与资源化利用。优化锅炉、窑炉燃烧工况，推进低氮燃烧改造与烟气治理设施提档升级。推动车船动力装备迭代更新，协同削减燃烧过程氧化亚氮排放。大

力推广测土配方施肥，落地施肥“三新”集成技术，推进秸秆综合利用，减少农田氧化亚氮释放。规范污水、固体废物处置流程，严控废弃物降解过程中的氧化亚氮逸散。

# (六)推进含氟气体分类管控

立足全省半导体、新型显示特色产业集群，探索实施含氟温室气体全链条减排治理。强化电力系统六氟化硫（ $\mathrm{SF}_6$ ）管控，不断削减  $\mathrm{SF}_6$  排放。严格落实氢氟碳化物生产与消费配额管理制度。

# 四、积极参与全国碳市场建设

# (七)深度融入全国碳排放权交易市场建设

立足我省重点排放行业发展实际，配合做好全国碳市场行业扩容工作。强化碳排放数据质量监管，完善省市联动的碳市场数据核实查证机制，加强碳排放数据数智化监管，依托大数据、人工智能等数字化技术，实现重点排放单位碳排放数据全过程管控。加强重点控排企业配额清缴动态监测与预警调度，指导企业依规完成配额履约清缴，持续提升碳市场日常监管与行政执法效能。充分发挥全国碳市场激励约束作用，压实企业温室气体减排主体责任，以市场化机制助推全省碳排放双控目标落地。

# (八)主动对接全国自愿减排交易市场建设

对标国家自愿减排交易市场制度建设要求，强化省内温室气体自愿减排项目全流程监管，建立并动态更新省级自愿减排项目储备库。全面摸排全省电力装备  $\mathrm{SF}_6$  在用存量，支持电力部门开发  $\mathrm{SF}_6$  回收净化自愿减排项目；鼓励两淮矿区立足自身优势，做好煤矿低浓度瓦斯资源化利用项目建设和开发。开展本地化减排方法学遴选和攻关，推动相关成果纳入国家方法学目录。积极拓展国家核证自愿减排量（CCER）应用场景，引导各类市场主体

使用 CCER 抵销碳排放，以市场化手段助推全省温室气体减排。

# 专栏2 碳市场数智化监管能力建设工程

(1) 碳排放权交易市场数智化监管能力建设。建设碳排放权交易市场数据质量分析预警平台，运用大数据、人工智能大模型等技术手段，加强对碳市场全链条数据的智能化分析与审核，提升对纳入全国碳排放权交易市场的重点行业企业碳数据质量问题的识别能力。  
(2）温室气体自愿减排项目监管服务能力建设。强化省内CCER项目全流程监管，依托数字化、智能化手段，实现CCER项目及减排量全流程跟踪、核查与监管。建立动态更新的省级CCER项目储备库，全方位服务省内CCER项目规范开发与落地。

# 五、统筹产品碳足迹全链建设与应用

# (九)推进产品碳足迹核算及数据库建设

聚焦煤炭、水泥、钢铁等基础能源与大宗原材料行业，组织开展本地化产品碳足迹因子测算工作，累计完成300个以上重点产品碳足迹核算。统筹建设省级重点产品碳足迹因子数据库，强化碳足迹数据安全与知识产权保护管理。依托数据库成果为省内外贸出口企业提供精准可溯源的产品碳足迹核算支撑，助力企业合规应对境外绿色贸易壁垒。支持科研院所、行业协会与重点企业协同参与碳足迹核算标准编制。

# (十)深化开展碳足迹标识认证工作

深化合肥、滁州产品碳足迹标识认证试点建设，因地制宜有序拓展试点覆盖区域与行业范围。坚持市场化导向，引导企业自愿开展碳标识认证，依规在产品、外包装及宣传物料上规范使用碳标识。依托省内高校、行业协会资源力量加强碳足迹标识认证业务培训，推动碳标识认证结果跨行业、跨领域互通互认，持续拓展碳足迹市场化应用场景与空间。

# 六、筑牢气候风险综合应对体系

# (十一)构建适应气候变化治理格局

建立气候变化观测预估、风险评估、适应行动、成效反馈、优化提升的全链条工作闭环。对标国家部署落地适应气候变化部门责任清单制度，建立健全省、市、县三级联动的适应气候变化工作责任体系。支持具备基础的地市深化气候适应型城市建设，择优争创国家级试点，扎实推进合肥、池州中国气候宜居城市提质建设。结合城市更新、老旧小区改造、乡村建设等重点工程统筹实施适应性改造，因地制宜打造各类气候适应型城乡微单元。

# (十二)健全气候风险监测评估机制

优化全省气候观测站点布局，完善立体化气候监测网络，提质升级寿县国家气候观象台，培育建设国家气候观象台与大气本底观测站点。依托全省生态质量综合监测站点资源，拓展气候变化驱动下生态变化专项监测，提升气候变化生态影响分析研判能力。加强生态敏感区、人口密集区气候变化风险监测预测，开展

重点行业领域气候风险动态评估，初步构建气候变化风险定期评估会商制度。在重大工程安全风险评估中，强化气候变化影响和风险评估，制定落实风险管控措施。研究编制00101省气候变化风险区划，为防灾减灾与适应性施策提供科学支撑。

# (十三)强化极端气候防灾减灾能力

加快建成极端天气新一代短临预警业务体系，健全政府统筹、多渠道协同的预警信息发布机制。加快补齐工程体系、基础设施等防灾减灾能力短板，实施全域水患治理工程，完善长江、淮河、新安江干流骨干防洪工程体系，开展山区河道、蓄滞洪区、病险水库等除险加固、工程治理，提高防洪排涝应急处置能力。优化大安全大应急框架下应急指挥机制，强化重大气象灾害应急处置能力。统筹灾后恢复重建与气候适应建设，科学引导人口、产业、基础设施向气候低风险区域合理布局，从源头降低灾害暴露度。

# 七、推进气候适应系统建设

# (十四)筑牢自然生态系统适应韧性

指导意见
完善全省水文监测站点布局，加快数字流域建设，构建互联互通、集约高效的现代化智慧水利体系。提升陆地生态系统灾害预警、防控与治理能力建设，统筹推进水源涵养区、河湖生态缓冲带、水域生境、水生生物群落系统性保护修复，不断筑牢水生态安全管护根基。系统开展生物多样性调查与保护成效评估，聚

焦扬子鳄、长江江豚、银缕梅、大别山五针松等本土珍稀濒危物种及其栖息地实施专项保护修复，持续提升生态系统气候适应韧性。

# (十五)增强经济社会系统适应能力

构建气候适应型国土空间开发保护格局，推动气候变化影响与灾害综合风险评估成果同国土空间规划信息深度融合落地。健全气候适应型粮食安全保障体系，立足皖北及沿淮片区打造粮食生产气候安全屏障，支撑“千亿斤江淮粮仓”建设；发展气候智慧型农业，选育气候适应性良种、完善农业气象灾害保险机制，增强农业气候适应韧性。常态化研判气候变化人群健康风险与区域脆弱性，针对沿江防洪骨干工程、池州宣城“西电东送”特高压廊道等关键基础设施防控极端气候隐患，把气候风险管理贯穿重大项目全生命周期。深化城市气候风险研判，推进城市气候风险地图编制并纳入城市体检工作清单，提升文旅、金融等敏感型二、三产业气候适应韧性。

# 八、夯实气候治理制度与核算支撑基础

# (十六)健全气候法规标准体系

贯彻落实《生态环境法典》，主动对接国家应对气候变化、碳达峰碳中和专项立法进程，在地方性法规和行政规范性文件立改废释中研究嵌入应对气候变化相关条款。加快探索减排与适应领域地方标准研制，落实国家关于温室气体管控、应对气候变化

纳入建设项目生态环境影响评价的相关要求。

# (十七)提升碳排放统计核算能力

建立省级地区碳排放年报、快报制度，夯实省市能源平衡表编制工作，压实各级统计数据归口责任。常态化摸排重点用能和碳排放单位煤炭消耗底数。推进省、市两级温室气体清单常态化编制，探索建立省级联审机制把关各市清单数据质量。建设省级温室气体排放清单数据库。规范非二氧化碳温室气体排放数据报送与核查工作。

# (十八)强化财税金融低碳支撑

积极争取“两新”“两重”专项资金，夯实应对气候变化财政保障根基，构建有利于应对气候变化的财政政策体系。落实碳减排相关税收优惠政策，发挥税收激励引导作用。完善气候投融资配套制度与标准体系，持续深化滁州国家级气候投融资试点建设，布局培育省级气候投融资试点，撬动更多社会资本投向降碳减排、减污降碳协同创新、气候适应建设等重点领域。引导金融机构创新绿色金融产品与服务，规范金融机构可持续信息披露管理，用好碳减排支持工具等货币政策工具，为合规绿色低碳项目提供低成本融资支撑。

# 专栏3 碳排放数字化管控与低碳示范创建工程

(1) 温室气体核算数智化建设工程。建设省级温室气体排放清单数据库，实现多源活动水平数据、排放因子数据的归集与规范化管理，支撑省级温室气体清单常态化、标准化编制，具备数据校核、统计分析、数据展示等智能化功能。搭建省级重点产品碳足迹因子数据库，实现产品碳足迹单元过程数据、模型数据的归集管理，推动与国家相关数据库互联互通。

# 九、推进低碳科创赋能与全民协同共治

加强气候变化关键过程与机理研究，研发气候变化影响和风险评估技术，构建气候变化风险早期预警体系；围绕绿色低碳转型开展前沿基础研究，夯实技术创新理论根基。依托合肥综合性国家科学中心能源与环境研究院、中科大碳中和研究院等高端科研载体，在绿色低碳领域布局建设省实验室、省产业创新研究院（省技术创新中心)等科创平台，积极培育争创国家级科创平台。聚焦煤炭清洁高效利用、规模化可再生能源开发等方向，系统开展低碳关键核心技术攻关；加快二氧化碳捕集、利用与封存（CCUS）技术研发，推进非二氧化碳温室气体减排治理与资源

化回收利用技术研发，重点攻关含氟气体源头减量替代、回收再生利用及末端降解销毁技术。大力推广低碳、零碳先进工艺，加快各类低碳技术成果转化及产业化示范应用。推动人工智能、大数据等数字技术深度嵌入研发、监测、管控全流程，全面提升应对气候变化全链条数字化、智能化支撑能力。
